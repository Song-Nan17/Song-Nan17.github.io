CSS基础笔记

CSS 层叠样式表，用于渲染HTML元素标签的样式。

选择器

除了几种常见的选择器外，还有：

  • A *选择A元素内的所有元素；
  • A + B选择所有直接跟在A元素后面的B元素；
  • A ~ B选择跟在A元素后面的所有B元素；
  • :first-child选择所有元素的第一个子级元素，如：p:first-child选择所有是第一个元素的p，div p:first-child选择所有div中的第一个p；
  • :only-child选择所有是父级元素唯一子元素的元素，如：span:only-child选择所有是唯一元素的span，ul li:only-child选择所有是ul中唯一元素的li；
  • :last-child选择所有元素的最后一个子级元素，如：span:last-child选择所有是最后一个元素的span，div p:last-child选择所有是div中最后一个元素的p；
  • :nth-child(x)选择有元素的第x个子级元素，如：:nth-child(8)选择所有第八个元素，div p:nth-child(2)选择所有是div中第8个元素的p；
  • :nth-last-child(x)选择所有元素的倒数第x个子级元素，如：:nth-last-child(2)选择所有的倒数第2个元素；
  • :first-of-type选择所有元素中的第一个某种元素，如：span:first-of-type选择所有的第一个span元素；
  • :nth-of-type(x)选择所有是type(x)的元素，如：div:nth-of-type(2)选择所有的第二个div元素，.example:nth-of-type(odd)选择所有的奇数个且类为example的元素；
  • :nth-of-type(An+B)从第B个开始选择所有第A个元素（包括第B个），如：span:nth-of-type(6n+2)从第2个开始数选择所有的第6个元素（选择包括一开始的第2个）;
  • :only-of-type选择在父级元素中是唯一一个该种类子级元素的子级元素，如：p span:only-of-type选择所有是p中唯一一个span的span；
  • :last-of-type选择在父级元素中是该种类子元素的最后一个的子级元素，如：div:last-of-type选择所有的最后一个div;
  • :empty选择所有没有子级元素的元素；
  • :not(X)选择所有非X的元素，如：div:not(:first-child)选择所有不是最后一个元素的div;
  • [attribute^="value"]选择所有属性值以value开头的元素;
  • [attribute$="value"]选择所有属性值以value结尾的元素;
  • [attribute*="value"]选择所有属性值包含value的元素;

    层叠

    CSS通过特异度解决块样式的冲突。
  • 简单选择器的特异度级别：
    • level 0 ：*
    • level 1 ：标签选择器、伪元素
    • level 2 ：类、伪类
    • level 3 ：id
    • level 4 ：内联
  • 引入方式的样式优先级：内联 > 嵌入 > 外链
  • 同一级别选择器，后写的覆盖先写的。
  • 有”!important”样式的，优先级最高。

    继承

  • 子元素会继承父元素的样式。
  • 子元素的样式优先级高于父级元素。

    盒模型

    从里到外由content, padding(透明), border, margin(透明)组成。
  • width和height指content box的宽和高。
  • 添加box-sizing:border-box属性后，weight/height = border + padding + content.
  • padding
    指内边距。
    • 缩写：padding: top值, right值, bottom值, left值;（上-右-下-左）
    • 缩写：padding: top&bottom值, right&left值;（上下-右左）
  • margin：
    指外边距。
    • 缩写：margin: top值, right值, bottom值, left值;（上-右-下-左）
    • 缩写：margin: top&bottom值, right&left值;（上下-右左）
  • margin折叠
    两个或多个普通文档流盒子垂直方向上的外边距会发生折叠。margin折叠是为实现文本段落间距而提供的特性，多数情况下，折叠垂直外边距可以在视觉上显得更美观。
    • 需要时相邻的非浮动元素
    • 折叠发生在垂直外边距上
    • 折叠后取其中最大的margin作为最终值
    • overflow:hidden;，会取消margin折叠

      overflow

      指的是溢出控制
  • 取值：visible | hidden | scroll | auto

    普通流

    norm flow，将窗体自上而下的分为一行一行，块级元素从上至下，行内元素在每行中从左到右地挨次排放。
  • 除根元素、浮动元素和绝对定位元素外，其它元素都在常规流之内(in-flow)
  • 而根元素、 浮动和绝对定位的元素会脱离常规流(out of flow)
  • 常规流中的盒子，属于块级格式化上下文或行级格式化上下文

    float

  • 浮动元素从常规流中脱离，被漂浮在容器(包含块)左边或右边
  • 浮动盒会一直漂到其外边缘挨到容器边缘或另外的浮动盒
  • 浮动元素不会影响其后面的流内块级盒
  • 但是浮动元素后面的行级盒子会变短以避开浮动元素

    clear

    指定元素哪一边不能与之前的浮动框相邻.
  • 取值：left | right | both

    position

  • static静态定位，元素出现在普通流中，不受top, left等影响。
  • fixed元素位置相对于浏览器窗口是固定位置，即使窗口滚动。eg: 广告。
  • relative元素相对于其正常位置定位。移动相对定位元素，但它原本所占空间不会改变。
  • absolute相对于最近的已定位父元素进行定位，元素不再占有父文档流位置。
  • z-index用于指定元素的堆叠顺序。
  • sticky基于用户的滚动位置来定位。
